这些是你在这个项目中实际运用过，且目前市场上非常吃香的前端技术：

## 2. 项目经验包装 (写在简历的“项目经历”模块)

不要只写“用 Vue 写了一个网站”，要用**“动作 + 技术 + 成果/亮点”**的 STAR 法则来写。你可以直接复制以下这几条项目亮点：

> **项目名称：** AI 生产力导航与场景化教程平台 (独立开发) **技术栈：** Vue 3 + TypeScript + Vite + Ant Design Vue + VueUse
>
> **项目描述：** > 独立设计并开发的一款面向大众的 AI 工具教程域网站。通过场景化分类与轻量级交互，降低普通用户的 AI 学习门槛。
>
> **核心工作与技术亮点：**
>
> - **现代前端架构搭建：** 基于 Vue 3 Composition API 与 TypeScript 构建单页面应用（SPA），保障代码的高可维护性与类型安全。
> - **动态 Markdown 渲染引擎：** 通过配置 Vite 插件 (`unplugin-vue-markdown`)，实现了 Markdown 文件到 Vue 组件的自动化编译，极大地提升了内容维护效率。
> - **混合组件注入（首创交互）：** 打破传统纯文本教程限制，实现在 Markdown 文本中直接挂载自定义交互组件。
> - **核心交互功能开发：** 引入 `@vueuse/core` 库，封装了带有状态反馈的 `PromptCard`（提示词卡片）组件，通过调用浏览器原生剪贴板 API 实现一键复制功能，优化用户体验。
> - **响应式 UI 与路由设计：** 深度定制 Ant Design Vue 组件库，覆盖默认样式以实现个性化主题。利用 Vue Router 动态路由 (`/category/:id`) 与本地数据映射，实现无刷新、秒加载的分类目录导览。
