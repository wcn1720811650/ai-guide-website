# 📝 “AI 枕边书” 产品与技术规划文档 v1.0

## 一、 产品定位与核心愿景

- **产品名称：** AI 枕边书（暂定）
- **Slogan：** 不讲算法，只讲怎么帮你提早下班。
- **目标用户：** 听过 AI 但不知道怎么用的职场白领、大学生、自媒体新手以及对新技术有焦虑的普通人。
- **核心痛点：** 现有的 AI 教程太硬核、看不懂；不知道怎么写提示词（Prompt）；缺少具体的应用场景。
- **核心解决方案：** 提供“场景化”的图文教程，配合“一键复制”的现成提示词卡片，让用户能“拿来即用”。

------

## 二、 核心功能路线图 (Roadmap)

我们将需求分为三个阶段（P0 代表最紧急必须做的，P2 代表未来锦上添花的）：

## 🟢 阶段一：MVP（最小可行性产品）- 跑通闭环 (P0)

*目标：能让用户顺利看完一篇文章并复制提示词。*

- **[✅ 已完成] 基础架构：** Vue 3 + TS + Vite 环境搭建。
- **[✅ 已完成] 页面骨架：** Ant Design 响应式布局（适配电脑和手机）。
- **[✅ 已完成] 核心交互：** `PromptCard` 一键复制提示词组件。
- **[⏳ 待开发] 内容渲染引擎：** 引入 Markdown 解析器，让你以后只需要写 `.md` 文件，网站就能自动把它变成漂亮的网页文章，而不需要你手写 Vue 代码。
- **[⏳ 待开发] 基础内容填充：** 上线前至少准备 3 篇高质量的核心教程（见第三部分内容规划）。

## 🟡 阶段二：体验升级 - 丰富网站结构 (P1)

*目标：让用户觉得这是一个专业的教程库，能自己找到想看的内容。*

- **分类导航系统：** 按“职场”、“生活”、“学生”等标签给文章分类，并在顶部导航栏提供入口。
- **全局搜索功能：** 接入简单的本地搜索，用户搜“周报”，就能跳出写周报的 AI 教程。
- **工具导航台：** 单独开一个页面，收录并评测目前主流的 AI 工具（如 ChatGPT, Claude, Kimi, 智谱清言等），带上直达链接。

## 🔴 阶段三：用户黏性与商业化试水 (P2)

*目标：留住用户，并探索变现可能。*

- **反馈机制：** 每篇文章底部增加“👍 有用 / 👎 没用”按钮，收集用户反馈。
- **暗黑模式：** 一键切换夜间护眼模式。
- **高级 Prompt 库：** 将特别优质的提示词打包，设置为“扫码关注公众号获取”或小额付费解锁。

## 四、 技术架构与后续开发指南

为了实现上面的需求，你的技术栈锁定如下，请妥善保存这份清单：

1. **前端框架：** Vue 3 (Composition API) + TypeScript
2. **构建工具：** Vite
3. **UI 库：** Ant Design Vue (负责整体排版、按钮、卡片、导航栏)
4. **路由管理：** Vue Router 4
5. **图标库：** Lucide Vue Next
6. **工具函数：** VueUse (负责一键复制等浏览器原生 API 的调用)
7. **【下一步重点】Markdown 渲染：** 使用 `vite-plugin-md` 或 `markdown-it` 将 `.md` 转换为页面。
8. **部署上线：** 后期代码写完，直接推送到 Github，使用 Vercel 或 Netlify 平台进行**免费**的一键自动化部署，生成你的专属网址。
